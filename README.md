This is guide for new Catholics to help them learn the Holy Rosary.  It assumes that the user knows the words to the repeated decade prayers,the Our Father, the Glory Be, and the Fatima Prayer.
The remaining prayers are more difficult to learn beacuse they are only said once during the whole rosary.
St. Michael's Prayer, which is not required, is included at the very end as it is often said at the conclusion of the Rosary by many Catholics.
The Mysteries are included with radio buttons to change Mystery sets in case the user wants to practice mysteries other than the those assigned to the actual calendar date.
Instructions are provided at the bottom of the Rosary telling the user how to save the to their iPhone homescreen.
The app requires internet connection to run properly.
The app can and should be saved as HTML and bookmarked in the case that internet is not available.
